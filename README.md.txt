# US Election Sentiment Analysis

This project performs a sentiment analysis on tweets about the U.S. 2024 election.

## Project Overview
- Uses Tweepy to fetch Twitter data.
- Preprocesses text data.
- Analyzes sentiment with TextBlob.
- Visualizes sentiment distribution and word clouds.

## Setup Instructions
1. Clone this repository.
2. Set up the `sentiment_analysis_us_elections` environment as detailed in the project steps.
3. Run the code in the `notebooks/analysis.ipynb` notebook to replicate the analysis.

## Dependencies
- Python 3.8 or later
- Tweepy, TextBlob, NLTK, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
